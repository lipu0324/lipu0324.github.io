# 指针、数组与自增表达式

## 核心概念

数组是一段连续存储的对象集合，数组名在多数表达式中会退化为指向首元素的指针。指针变量保存地址，可以被重新赋值，也可以通过指针算术移动到相邻元素。数组对象本身不是可修改左值，因此不能执行 `arr++`，但指针变量可以执行 `p++`。

指针和数组都可以使用下标访问：

```cpp
p[i] == *(p + i)
arr[i] == *(arr + i)
```

但 `sizeof(arr)` 和 `sizeof(p)` 不同：

```cpp
int arr[4];
int* p = arr;

sizeof(arr); // 4 * sizeof(int)
sizeof(p);   // 指针大小，64 位系统通常是 8 字节
```

## 前置自增与后置自增

前置自增：

```cpp
int x = 5;
int y = ++x;
```

含义：

```text
x 先变成 6
y 得到 6
```

后置自增：

```cpp
int x = 5;
int y = x++;
```

含义：

```text
y 得到旧值 5
x 之后变成 6
```

规则：

- `++x`：先自增，表达式值是新值。
- `x++`：表达式值是旧值，副作用是之后自增。

## 高频指针表达式

```cpp
int arr[] = {10, 20, 30, 40};
int* p = arr;
```

```cpp
(*p)++;
```

修改 `p` 当前指向的值，`arr[0]` 从 `10` 变成 `11`，`p` 不移动。

```cpp
*p++;
```

等价于：

```cpp
*(p++);
```

先取旧指针指向的值，再让 `p` 移动到下一个元素。

```cpp
*++p;
```

等价于：

```cpp
*(++p);
```

先让 `p` 移动到下一个元素，再解引用。

## 示例

```cpp
int a[] = {1, 2, 3, 4};
int* q = a;

int x = *q++;
int y = ++*q;
int z = *++q;
```

执行过程：

```text
x = 1
y = 3
z = 3
a = {1, 3, 3, 4}
q 指向 a[2]
```

## 越界访问

如果指针移动到数组范围之外，再解引用属于 undefined behavior。程序可能读到异常值、崩溃，也可能看似正常运行，但语义已经不可靠。

```cpp
int a[3] = {1, 2, 3};
int* p = a + 3; // one-past-the-end 指针可以存在
*p;             // undefined behavior
```
